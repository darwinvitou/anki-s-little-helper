# Gemini AI Helper for Anki

A custom Anki add-on that embeds a web browser directly into the card review screen. This allows you to seamlessly interact with Gemini AI to ask questions, get explanations, or look up concepts without ever leaving your study session.

## Features
* **Embedded Browser:** No need to switch between windows; the browser lives inside Anki.
* **Instant AI Help:** Keep Gemini open alongside your cards for quick lookups during reviews.

## Installation
1. Open Anki
2. Go to Tools -> Add-ons
3. Click on View Files
4. Extract the content into the folder
5. Restart Anki
